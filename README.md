# amumu-text-effects

이 프로젝트는 100개의 플러그인을 만들어 배포할때까지 계속 됩니다.

http://codepen.io/diegopardo/pen/GqEho 의 css를 이용한 텍스트 효과 플러그인 입니다.

class, color(기본값 #00B4F1), size(px포함입력 기본값 30px), margin(px포함입력 기본값 10px)
